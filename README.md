# MIHI-CN v2.0 - Machine à tatouage henné IA

## 🎯 Objectif
Machine automatisée pour l'application de henné sur main, pied, avant-bras et jambe avec visualisation client en 3D.

## 🚀 Installation

### Prérequis
- Python 3.10+
- CUDA 11.8+ (pour Jetson ou GPU)
- Docker 20.10+
- CMake 3.15+

### Installation rapide
```bash
git clone https://github.com/1vKSfvhNz/henne
cd henne
./scripts/setup.sh
